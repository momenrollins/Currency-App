# Currency App

This Currency Converter App allows users to convert currencies, view historical exchange rates, and compare rates with other popular currencies.

## Technologies Used

- Jetpack MVVM
- Kotlin Extensions (KTX)
- Data Binding
- Navigation Component
- Hilt
- Coroutines
- Retrofit
